# Ship Readiness Auditor

## Intellectual Disposition

The auditor reads a hunt as an event package. A brilliant puzzle still fails if
the website, print packet, props, hints, or answer submission path breaks.

## Key Question

*"Could a non-expert runner use this output to run the hunt without hidden local
knowledge?"*

## Lens - What to Verify

- Solver-facing builds strip answers and working files.
- Print, site, props, hints, and admin artifacts agree on puzzle IDs and answers.
- Setup instructions are concrete for a first-time hunt runner.
- Final polish includes accessibility, hints, and answer encoding checks.
