## SOURCE 15: Water Beds (PHA / Placement Test)

**URL**: https://puzzle.university/puzzle/ph10.html and solution https://puzzle.university/solution/ph10.html
- **Author**: Dana Young
- **Original hunt**: Microsoft Puzzlehunt A / 10 (2007)
- **In PH23 context**: Placement Test feeder
- **Type**: Nonogram / aquarium water-level logic puzzle
- **Mechanic**: Grid-based; shade cells for water; water level in each aquarium must be same across its full width; numbers indicate filled cells horizontally/vertically; letters hidden in cells "float" upward based on water level; read floating letters left to right, top to bottom
- **Answer**: THALASSOPHOBIA (= "THE FEAR OF DEEP WATER", which the floating letters spell)
- **Is it a rebus?**: No. It is a constrained logic puzzle with letter-floating extraction.

## SOURCE 18: Numerator, Denominator, Fascinator (PH7 / Placement Test)

**URL**: https://puzzle.university/puzzle/ph7.html and solution https://puzzle.university/solution/ph7.html
- **Author**: Dana Young
- **Original hunt**: Microsoft Puzzlehunt 7 / Alice in Puzzlehunt (2004)
- **In PH23 context**: Placement Test feeder
- **Type**: Hat-identification math puzzle
- **Mechanic**: Image shows characters wearing hats; each character's name replaced by sum of word lengths in their name; substitute sums into mathematical formula: [(17+5)(8+12)(22-7) - (9-4)(16-6)(14-10)] / [(14+10)(7-5)(12-6) - (10-5)(14-6)(12-5)]; result = 800; in calculator font, 800 looks like "BOO"
- **Answer**: BOO

## Pyramid Schemes — Detailed Entry (SOURCE 4)

**URL**: https://puzzle.university/puzzle/pyramid-schemes.html and solution https://puzzle.university/solution/pyramid-schemes.html
- **Author**: Jonah Ostroff (confirmed by solution page)
- **Round**: Economics
- **Solves**: 123
- **Type**: Criss-cross word grid with infinite downward pyramid propagation
- **Mechanic**:
  - Standard criss-cross grid with one given letter (C)
  - Grid's bottom two answers feed into copies below it → infinite pyramid structure
  - Each grid solves uniquely from letters inherited from parent grids
  - Numbered squares in solution to original grid read "FIFTH LAYER"
  - Solvers work downward through 5 pyramid layers, completing 7 unique grid versions
  - Read marked squares in fifth layer in "triangular answer" format
- **Answer**: DELTA
- **Word list includes**: AMUSE, AIRFIELD, ELEGIACAL, EVENT, ARABITOL, IRRITANTS, IONIC, DAYLIGHT, LEGIONARY, LEASE, PISCINAE, SOLITAIRE, PROTISTA, SVENGALIS, SATURNIC (and more)
- **24 total words**: 16 for meta-logic propagation across layers, 8 for extraction, 12 additional for grid connectivity
- **Design notes**: Inspired by Kevin Wald's "The World's Tallest Cryptic" and "Curious Customs"; originally conceived as 9-letter answer titled "Pascal's Triangle"; refined to current format

**AUTHORSHIP CLARIFICATION**: Pyramid Schemes was written by JONAH OSTROFF, not Kenny Young. Kenny Young wrote the Economics Meta and several other Economics round puzzles.

## Macroeconomics — Detailed Entry (SOURCE 9)

- **Author**: Kenny Young
- **Round**: Economics
- **Solves**: 131
- **Type**: C++ code obfuscation / programming puzzle
- **Mechanic**:
  - C++ source code with $ placeholder characters throughout
  - Replace each $ with a lowercase letter so program compiles with zero warnings
  - Constraint: no #define 'word' begins with same letter it is replacing
  - Program must output an English word of 5 or more letters
  - Code uses macro definitions (Z() and A() functions), #include statements with obfuscated filenames, extensive macro replacements, struct M with template implementation, function G() with vector operations and character manipulation
