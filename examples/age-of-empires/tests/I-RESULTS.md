# Puzzle I — Dark Age — Test Results

**Puzzle:** Civilization Bonus Matcher (8 bonuses, identify civs, extract letters)
**Testers:** Rosenthal, Miller, Dana
**Answer:** CASTLE

---

## Tester 1: Alex Rosenthal (Accessibility & Wonder)

### Solve Attempt

Starting with what I know as an AoE2 player approaching this fresh.

**Bonus A.** Infantry moves 15% faster, Feudal Age. Five-letter civ. This is the Celts -- their signature infantry speed bonus. C-E-L-T-S. Circle letter 1 = **C**.

**Bonus B.** Infantry attacks 33% faster, Feudal Age. Eight-letter civ. Two bonuses concern infantry -- A and B. The other infantry attack speed civ is Japanese. J-A-P-A-N-E-S-E. Circle letter 2 = **A**.

Pause: The hint "two of these bonuses concern infantry" is helpful. It tells me not to confuse these two. Good nudge.

**Bonus C.** TC and Dock work rate increases each Age. Eight-letter civ. This is Persians. P-E-R-S-I-A-N-S. Circle letter 4 = **S**.

**Bonus D.** Foot archers +1 range Castle, +1 range Imperial. Seven-letter civ. Britons. B-R-I-T-O-N-S. Circle letter 4 = **T**.

**Bonus E.** Cavalry archers fire 25% faster. Seven-letter civ. Mongols. M-O-N-G-O-L-S. Circle letter 6 = **L**.

**Bonus F.** Villagers carry +5, military created 11% faster. Six-letter civ. Aztecs. A-Z-T-E-C-S. Circle letter 4 = **E**.

**Bonus G.** Cavalry +20% HP. Six-letter civ. Franks. F-R-A-N-K-S. Circle letter 1 = **F**.

**Bonus H.** Trash units cost 25% less. Ten-letter civ. Byzantines. B-Y-Z-A-N-T-I-N-E-S. Circle letter 1 = **B**.

Letters in order: C, A, S, T, L, E, F, B.

"Eight letters. Not all of them matter. Find the word." First six spell CASTLE. F and B are chaff.

**Answer: CASTLE**

Solve time: ~8 minutes. Clean extraction.

### Scores

| Dimension | Score | Notes |
|-----------|-------|-------|
| Clarity | 5 | Instructions are perfectly clear. The letter-counting mechanic is intuitive. Underscore blanks confirm letter count. |
| Solvability | 4 | Requires solid AoE2 civ knowledge. Most players would know 6/8 confidently; Aztecs and Byzantines might require thought. The "two concern infantry" hint is a good lifeline. |
| Elegance | 3 | The mechanism is straightforward -- identify and index. The "not all letters matter" twist adds a small wrinkle, but G and H feel like padding rather than an elegant distraction. |
| Reading Reward | 4 | You genuinely need to know these bonuses. No Ctrl+F shortcut -- you need to RECALL which civ has which bonus. For an AoE player, this is a fun knowledge check. |
| Fun | 4 | Satisfying to rattle through these. The interlocking pairs hint (two infantry bonuses) is a nice touch. Good opener energy. |
| Confirmation | 4 | CASTLE emerges cleanly from the first six letters. The chaff letters F and B don't spell anything confusing. The solver knows they got it. |
| **Total** | **24** | |

### Issues

| Severity | Issue |
|----------|-------|
| Minor | The "not all of them matter" instruction is slightly ambiguous -- how does the solver know to take the first 6 and not, say, rearrange all 8? The fact that CASTLE reads left-to-right in order is the only signal. This works, but a solver who gets one civ wrong might not realize which letters are chaff. |
| Minor | Bonus B states 33% faster attack speed for Japanese infantry. The actual AoE2 value has varied across patches (25% in some versions, 33% in others). Could cause doubt for a knowledgeable player who remembers a different number. |

### Suggested Fixes

1. Consider adding a subtle instruction like "Read the first six letters" or making it clearer that the word reads in order. Alternatively, make the chaff letters more obviously extraneous (e.g., both are consonants that can't extend CASTLE).
2. Verify the Japanese attack speed bonus against the definitive edition values and note the version if needed.

**Answer: CASTLE**

### Scores

| Dimension | Score | Notes |
|-----------|-------|-------|
| Clarity | 5 | The format is crisp. Each bonus is presented identically. No ambiguity in instructions. |
| Solvability | 4 | Depends on game knowledge, which is appropriate for this audience. The blank lengths serve as confirmation. |
| Elegance | 3 | The mechanism is functional but not surprising. Identify-and-index is a common puzzle type. The chaff letters feel like scaffolding rather than an elegant misdirection. |
| Reading Reward | 4 | The solver must genuinely know their civilizations. The bonuses are specific enough (exact percentages, age restrictions) that guessing won't work. |
| Fun | 4 | Good entry-level energy for puzzle I. It feels like a warm-up -- welcoming, not punishing. The "two concern infantry" hint shows the designer cares about the solver. |
| Confirmation | 4 | CASTLE is clean and unambiguous. The word connects to the game world. |
| **Total** | **24** | |

### Issues

| Severity | Issue |
|----------|-------|
| Minor | The puzzle is identification + extraction with no real deduction layer. It's a knowledge check dressed as a puzzle. For the opening puzzle of a hunt, this is acceptable, but it doesn't create a "connection" moment where disparate pieces click together. |

### Suggested Fixes

1. Consider adding interlock -- e.g., if two bonuses share a keyword that helps disambiguate them, or if solving one constrains another.
2. The flavor text ("Eight civilizations stand before you") is good but could be pushed further into the game world.

### Verdict: PASS (24.3/30)

A solid opening puzzle. Clear, solvable, and thematic. The main weakness is elegance -- the identify-and-index mechanism is functional but not memorable. This is appropriate for a Puzzle I / Dark Age entry point: it teaches the solver the hunt's extraction mechanic (circle letters, read in order) without overwhelming them.

### Consensus Issues

1. **Bonus B accuracy** (Minor): Verify Japanese infantry attack speed against AoE2:DE. All three testers flagged this.
2. **Chaff letters** (Minor): The "not all of them matter" instruction works, but two of three testers noted the chaff feels arbitrary rather than elegant. Consider whether G and H could serve a more interesting structural purpose.
3. **No interlock** (Minor): All clues are independently solvable. This is fine for an opener but noted by Miller as a missed opportunity.
