# External Puzzle Reference Library

Real puzzle hunt puzzles collected for research use. These are publicly available puzzles from credible hunts, used as external validation data for the paper "Expert AI Panels for Creative Work Evaluation: Evidence from Puzzle Hunt Design" (ICCC 2026).

## Usage

These puzzles were classified by an AI expert panel (Dan Katz, Thomas Snyder, Dana Young profiles) and then subjected to an 8-condition ablation study. Results are in:
`C:\src\craftworks\research\games\games-ai-expert-panel-creative\results\`

## Selected for ablation (5-5-5 corpus)

**MIT/Elite (5):** Scicabulary, People Watching, Bridge Building, You're Telling Me, Information Relay
**Standard Hunt (5):** H2No, Dropypasta, Front and Center, Characters, What's Next?
**Games Magazine (5):** GM-01 through GM-05
