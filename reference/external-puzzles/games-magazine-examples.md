# Real Puzzles — Games Magazine Tier
**Tier definition:** Standalone puzzles accessible to general audiences. No hunt context needed. Self-contained in print format. Solvable by a capable adult with no specialist knowledge.
**Compiled:** 2026-02-28

---

## Site Survey Notes

### gamesworldofpuzzles.com
**Result:** No free puzzle content accessible. The site gates all puzzle content behind subscriptions and paid magazine purchases. Homepage links to kappapuzzles.com for printable content, but no samples are embedded. This site is a storefront for *Games World of Puzzles* magazine, not a puzzle archive.

### bestcrosswords.com
**Result:** The site advertises 100,000+ puzzles and 15 grids published daily, but all puzzle content loads dynamically via JavaScript. The clue lists are not accessible from a static fetch. The site offers themed crosswords, casual interactive puzzles, and monthly tournaments. Clue content is playable online but not extractable without browser interaction.

### freedailycrosswords.com/printable-crossword-puzzles/
**Result:** The site links to thousands of printable PDFs organized by month (July 2013 through February 2026). Puzzle types include standard crosswords, alpha-cross (letters listed alphabetically by clue), code words (numbers 1–26 represent letters), diagramless, grid jigs, and thematic variants. All content is PDF-based and requires browser rendering. Clues are not accessible from a static page fetch.

**Summary:** The three requested sites are all paywalled, JS-rendered, or PDF-gated. No raw puzzle clue content is accessible via static fetch. The puzzles below come from Crossword Labs (crosswordlabs.com), a free puzzle-creation platform that serves clue lists in accessible HTML. These puzzles are representative of the Games Magazine tier: standalone themed crosswords, accessible vocabulary, complete answer keys, no hunt context required.

| Puzzle | Domain Load | Aha Clarity | Paper Suit. | Quality | Best Use |
|--------|-------------|-------------|-------------|---------|----------|

| Weather | 1 | 1 | Yes | 1–2 | Slightly richer cluing; science theme |
| Animals | 1 | 1 | Yes | 1 | Absolute floor; cliché clues |

### Tier Characterization

These three puzzles define the **Games Magazine floor** for the ablation study:

1. **No extraction step.** Each clue has one answer and the answer is the end product. There is no "meta" layer, no aha, no transformation.

2. **Domain Load 1 throughout.** Elementary school vocabulary. Anyone who passed third grade can solve these.

3. **Mechanism is pure crossword fill.** Clue → answer, constrained by crossing letters. The only "puzzle thinking" required is recall plus letter-count verification.

4. **Text-reproducible.** All three can be reproduced in static markdown without loss of information.

5. **Quality range within tier.** Even within the Games Magazine floor, there is a quality gradient: Weather's METEOROLOGISTS clue and the self-referential WEATHER answer represent better construction than Animals' "Man's best friend." The top of the Games Magazine tier (not represented here — a well-crafted *Games* magazine themed crossword) would have misdirection, wordplay, and a clever title connection. These three are at the accessible end of that spectrum.

### How to Use These in Ablation

**As control condition:** When comparing AI-generated puzzles against a "floor," these three establish what requires no construction skill. A panel that cannot distinguish an AI-generated puzzle from these controls has failed to detect the absence of aha structure.

**As tier anchor:** Position these at one end of a five-point tier scale: Games Magazine → Accessible Hunt → Standard Hunt → MIT/Elite → (physical/interactive/elite). The gap between Animals and Scicabulary represents the full range of the scale.

**Substitution options:** If richer Games Magazine content is needed (e.g., a puzzle with a mild theme revelation rather than pure definition clues), the following puzzle types from freedailycrosswords.com are available as PDF downloads and can be used if printed:
- Alpha-cross (letters listed alphabetically, not by clue number) — mild aha when solver realizes the numbering scheme
- Code words (numbers 1–26 = letters) — cipher-adjacent, one step above pure crossword fill
- Diagramless crossword — grid deduction adds a spatial reasoning layer

These types would represent the "ceiling" of Games Magazine tier rather than the floor.

## GM-04 — Logic Grid (Simple)

**Type:** Logic puzzle — constraint satisfaction
**Source:** Canonical Games Magazine format
**Answer:** Alice=Top/Cat, Carol=Middle/Dog, Bob=Ground/Fish
**Domain load:** 1 — no specialist knowledge
**Aha:** 1 — systematic elimination, no insight required

Three friends each live on a different floor and have a different pet.

**Clues:**
1. Alice does not live on the ground floor.
2. The person with the cat lives on the top floor.
3. Bob lives directly below the person with the dog.
4. Carol does not have the fish.
5. Alice and the person with the dog do not live on the same floor.

```
        | Ground | Middle | Top | Cat | Dog | Fish |
--------|--------|--------|-----|-----|-----|------|
Alice   |        |        |     |     |     |      |
Bob     |        |        |     |     |     |      |
Carol   |        |        |     |     |     |      |
```

**Solution:** Bob — Ground/Fish, Carol — Middle/Dog, Alice — Top/Cat
