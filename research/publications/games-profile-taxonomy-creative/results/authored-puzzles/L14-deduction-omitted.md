# Condition L14 — Leave Out: Multiple Deduction Paths

## Puzzle

Each clue below describes a unit, technology, civilization, or map from Age of Empires II.
Identify the item named by each clue. The first letter of each answer, read top to bottom, spells a five-letter word.

---

**1.**
This siege weapon must unpack before firing.

**2.**
This Imperial Age siege unit can destroy trees.

**3.**
This infantry unit moves faster than any other infantry in the game.

**4.**
This unit is the Castle Age upgrade of the Skirmisher.

**5.**
This Monastery technology allows Monks to convert buildings.

---

## Solution Key

T: Trebuchet
O: Onager
W: Woad Raider
E: Elite Skirmisher
R: Redemption

Answer: TOWER

## Omission Notes

Each clue contains exactly one identifying fact with no redundant constraints — a solver who knows that fact reaches the answer immediately, but a solver who does not has no fallback. The "Multiple Deduction Paths" principle was omitted: no clue provides two or more independent routes to the same answer.
