# World — Age of Empires Reference

The verified knowledge base for the Wololo puzzle hunt. Every fact a puzzle uses must be traceable to this directory. Authors reference these files, not memory.

## Data Files

| File | Contents | Used by |
|------|----------|---------|
| `civs.md` | Civilization bonuses, unique units, team bonuses, wonders | Puzzle I |
| `units.md` | Unit stats, counter relationships, upgrade paths | Puzzle II |
| `techs.md` | Technology tree, prerequisites, costs, effects | Puzzle III |
| `maps.md` | Standard map types, terrain, resource spawns | Puzzle IV |
| `economy.md` | Gather rates, build times, age-up costs | Puzzle V |

## Rules

1. **All puzzle facts must appear in a world/ file.** If a puzzle claims "Britons get +1 range on archers," that fact must be in `civs.md`.
2. **Primary source: AoE2 Definitive Edition** (latest patch). Other versions noted where different.
3. **Authors verify** before using a fact. If uncertain, add a `[VERIFY]` tag.
4. **No invented facts.** Everything must be real AoE game data.
