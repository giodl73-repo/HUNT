# THE SIXTEENTH SHARD — Hints

3-tier hints for all 36 puzzles + 4 metas. Hoid's voice.

- **Tier 1 (nudge):** Points toward the right approach.
- **Tier 2 (direction):** Explains what to look for.
- **Tier 3 (near-solution):** Gives the specific mechanism.

---

## Round 1: Scadrial

### P01 — Burning Through

- **Tier 1:** Each description names a power. The power names a metal.
- **Tier 2:** The metals have numbers. Certain words in each description carry extra weight. Look for formatting.
- **Tier 3:** The bold word in each metal's description has a first letter. Order bold words by metal number. Read the acrostic.

### P02 — Alloy Partners

- **Tier 1:** Metals come in pairs. Base and alloy. Partners.
- **Tier 2:** Each pair produces a sentence. The sentences begin with letters that matter.
- **Tier 3:** Pair all 8 base metals with their alloys. Read the first letter of each pairing sentence in table order.

### P03 — Feruchemical Reserves

- **Tier 1:** The descriptions are Feruchemical, not Allomantic. Different table.
- **Tier 2:** Identify which metal stores which attribute. Then look at how the descriptions begin.
- **Tier 3:** Each Feruchemical identification maps to a metal. The sentence-initial letters of the six descriptions, ordered by metal number, spell the answer.

### P04 — The Allomantic Table

- **Tier 1:** The 4x4 grid has rules. Columns and rows carry meaning.
- **Tier 2:** Place metals by quadrant and internal/external classification. The constraints narrow the grid.
- **Tier 3:** Solve the grid placement. The acrostic is scrambled — unscramble from the placed metals to read the answer.

### P05 — Three Arts, One Spike

- **Tier 1:** Three descriptions per metal. One is Allomantic, one Feruchemical, one Hemalurgic. But one lies.
- **Tier 2:** The "lie" in each triple belongs to a neighboring metal's art. Identify the true triples.
- **Tier 3:** Once all 8 metals are correctly assigned across all three arts, read the intro-word acrostic in table order.

### P06 — The Coppercloud

- **Tier 1:** This is a logic grid. Build it systematically.
- **Tier 2:** The Seeker's report tells which seats pulse and which are hidden. Start from the Coppercloud at seat 4.
- **Tier 3:** Resolve all 8 seats. The first letters of the names in seats 1 through 6, reading left to right, spell the answer.

### P07 — The Lord Ruler's Thousand Years

- **Tier 1:** Events from the Final Empire. Put them in order.
- **Tier 2:** The chronological sequence starts before the Final Empire and ends after it falls.
- **Tier 3:** Order all 10 events earliest to latest. Read the first letter of each event description top to bottom. A six-letter word hides within the ten-letter string.

### P08 — Kelsier's Crew

- **Tier 1:** Each portrait describes one of Kelsier's eight. The number in the portrait matters.
- **Tier 2:** Identify each crew member. The Mistings have street names. The number indexes into the street name.
- **Tier 3:** Find each Misting's street name (Lurcher, Coinshot, etc.). Use the portrait's number to extract one letter per Misting. Skip Mistborn and non-Allomancers.

### P09 — The Social Ladder

- **Tier 1:** Six classes of the Final Empire. Rank them lowest to highest.
- **Tier 2:** The twelve souls help identify which paragraph describes which class. Sort the classes by rank.
- **Tier 3:** Once ranked lowest to highest, read the first letter of each paragraph in that order.

### P10 — The Survivor's Legacy

- **Tier 1:** Eight causes in Era 1. Eight effects in Era 2. They connect across three centuries.
- **Tier 2:** Match each cause to its effect. Then find the connection word from the bank that names the bridge.
- **Tier 3:** Order matched pairs by the chronological date of the Era 1 cause (earliest first). Read first letters of connection words top to bottom. The answer hides in the string.

### P11 — The Siege of Luthadel

- **Tier 1:** Multi-variable logic grid. Characters, roles, gates, metals.
- **Tier 2:** Start with the constraints that fix a single variable. Build outward.
- **Tier 3:** Resolve all variable assignments. The first letters of Kelsier's crew members' chosen options, in gate order, spell the answer.

### P12 — The Roughs Justice

- **Tier 1:** Twinborn have two abilities — one Allomantic, one Feruchemical. Identify both per character.
- **Tier 2:** The Allomantic quadrant and the Feruchemical metal together produce a letter.
- **Tier 3:** For each character, the Allomantic quadrant number (Physical=1, Mental=2, Temporal=3, Enhancement=4) indexes into the Feruchemical metal name.

---

### META-I — THE WELL

- **Tier 1:** The Allomancy table is more than a reference. It is the extraction device.
- **Tier 2:** Each puzzle maps to a metal. Place all 12 answers in the table. Pulling metals are the signal. Pushing metals are noise.
- **Tier 3:** For each Pulling metal with an answer, extract the letter at the quadrant depth (Physical=1, Mental=2, Temporal=3, Enhancement=4). Read in table order.

## Round 2: Roshar

### P13 — Herald's Madness

- **Tier 1:** Identify the Herald from each description. Each Herald has an Order and a gemstone.
- **Tier 2:** The ward number tells which letter to take — not from the Herald's name, but from the gemstone's name.
- **Tier 3:** Identify all 6 Heralds. Map each to their gemstone. Index into the gemstone name at the ward number.

### P14 — Ideals and Oaths

- **Tier 1:** Each blank is a word from a published Radiant Oath. Fill them in.
- **Tier 2:** One position in each missing word is marked with brackets. That position gives a letter.
- **Tier 3:** Fill all blanks. Extract the bracketed letter from each missing word. Order by Ideal number first, then by Order number within the same Ideal.

### P15 — Spren Bonds

- **Tier 1:** Each vignette describes a type of Radiant spren by behavior, not by name.
- **Tier 2:** The glyph dots mark a position within the spren type name. Identify the spren, then index.
- **Tier 3:** Match behaviors to spren types (Honorspren, Highspren, Ashspren, Inkspren, etc.). The dot count indexes into the type name.

### P16 — Radiant Roster

- **Tier 1:** Twelve figures. Not all are Radiants. Only the bonded ones contribute.
- **Tier 2:** Each description has one bolded word. Only the Radiants' bold words matter.
- **Tier 3:** Identify which figures are true Radiants. Sort by Order number. Read the first letters of their bold words.

### P17 — The Double Eye

- **Tier 1:** The 10 Orders sit on a ring. Each shares a Surge with its neighbor.
- **Tier 2:** Five positions are "infused" with a depth value. The depth tells which letter of the Order name to take.
- **Tier 3:** Place all 10 Orders on the Double Eye ring. For the 5 infused positions, extract the letter at the given depth from the Order name. Read clockwise from the Soldier's seat.

### P18 — The Surge Wheel

- **Tier 1:** Start from the anchor (Windrunners). Each Order shares one Surge with the next.
- **Tier 2:** Chain outward: Windrunners' second Surge is Gravitation. The next Order shares Gravitation. That is Skybreakers.
- **Tier 3:** Complete the chain for all 10 Orders. Seven positions are marked. Read those letters to spell the answer.

### P19 — Singer Forms

- **Tier 1:** Eight forms, eight testimonies. The trait-word matters more than the form name.
- **Tier 2:** Identify each Singer's form from the description. The bolded trait-word holds the letter.
- **Tier 3:** Index into each trait-word at the given position. Read in testimony order (1 through 8). The answer hides in the string.

### P20 — Bridge Four

- **Tier 1:** Scrambled names of Bridge Four members. Some carry stowaways.
- **Tier 2:** Unscramble each name. Six of nine have one extra letter that does not belong to the name.
- **Tier 3:** The six extra letters, read top to bottom, spell the answer.

### P21 — Rhythm of Answers

- **Tier 1:** Singer Rhythms carry emotional content. The bolded tone-word holds the letter.
- **Tier 2:** Identify each Rhythm from its description. Index into the tone-word at the given position.
- **Tier 3:** Extract from tone-words in order. The answer hides in the resulting string.

### P22 — The Chasms Below

- **Tier 1:** Every clue has two readings. The surface reading is Rosharan. The cryptic parse follows standard rules.
- **Tier 2:** Look for hidden words, charades, anagrams, and acrostics. Every answer is a Rosharan term.
- **Tier 3:** Solve all six cryptic clues. The circled letter in each answer, read top to bottom, spells the answer.

### P23 — Fabrial Workshop

- **Tier 1:** Each fabrial effect connects to an Essence. Each Essence belongs to a gemstone.
- **Tier 2:** Match the effect to its Essence (wind, fire, stone, etc.). The Essence names the gemstone.
- **Tier 3:** Extract from each gemstone name at the given position. Read in order.

### P24 — Stormwall

- **Tier 1:** Storms move across Roshar. Calculate when each storm reaches each watchtower.
- **Tier 2:** Each watchtower bears a signal letter. The storm arrival order determines the reading order.
- **Tier 3:** Sort watchtowers by storm arrival time. For ties, east before west. Read signal letters in that order.

---

### META-II — THE OATHS

- **Tier 1:** Not all Orders contribute. Only those who have sworn deeply enough.
- **Tier 2:** Count each Order's documented Ideals. Orders with 3 or more contribute. The Ideal count indexes into the answer.
- **Tier 3:** Five Orders have 3+ documented Ideals. For each, the Ideal count is the extraction position in that puzzle's answer. Read in Order number sequence.

## Round 3: The Cosmere

### P25 — Shattered and Whole

- **Tier 1:** Sixteen fragments. Each is a Shard. The Intents have been removed.
- **Tier 2:** Identify each Shard's Intent from its description. Six fragments carry bracketed extraction positions.
- **Tier 3:** Index into each marked Intent name at the given position. Read from fragment D through P in marked order.

### P26 — Vessel Roll Call

- **Tier 1:** The Vessels are named. The Shards' Intents hold the letters.
- **Tier 2:** Identify which Vessel holds which Shard. Six descriptions carry extraction markers on the Intent name.
- **Tier 3:** Extract from Intent names (not Vessel names) at the marked position.

### P27 — Odium's Kill List

- **Tier 1:** Five Splinterings. Four by Odium, one self-inflicted. The order matters.
- **Tier 2:** Identify each Splintered Shard. Determine the chronological order of Splintering — Odium's kills first.
- **Tier 3:** Read the first letter of each manner-of-Splintering phrase in chronological order. Alphabetical by Intent breaks ties within the same campaign.

### P28 — Investiture Types

- **Tier 1:** The magic systems have names. The names have letters.
- **Tier 2:** Identify which Shard's Investiture each description references. Extract from the magic system name, not the Shard name.
- **Tier 3:** Six magic system names (Awakening, Surgebinding, Feruchemy, etc.) each have a marked position. Extract and read in order.

### P29 — The 4x4 Shard Grid

- **Tier 1:** Place 16 Shards on a 4x4 grid. The axes have labels.
- **Tier 2:** The answer is not in the Shard names. It is in the axis labels.
- **Tier 3:** The column initials and row initials spell the answer. Read the axis labels.

### P30 — The Shattering

- **Tier 1:** A 16-seat circle. Ten known Vessels. Fourteen clues. Only one arrangement works.
- **Tier 2:** Start from the anchor: Clue 12 places Aona at seat 1. Clue 1 places Skai at seat 2. Build outward from there, using the proximity and separation clues to fix each Vessel's position.
- **Tier 3:** From the anchor: Aona(1), Skai(2), Rayse(4) via clue 4. Then use clues 9-11 to place Leras, Ati, and Edgli in the 5-10 range. Clue 6 fixes Tanavast. Clues 3, 5, 10, 13 fix Bavadin, Koravellium Avast, and Ulas Dal. Extract from the six marked seats.

### P31 — The Wanderer's Aliases

- **Tier 1:** Some encounters feature Hoid. Some feature other worldhoppers. Only one set of footprints matters.
- **Tier 2:** Determine which encounters feature Hoid specifically. His tells: he tells stories, he disguises himself, he collects abilities.
- **Tier 3:** For each Hoid encounter, the alias or role he takes has a marked letter. Read from the Hoid encounters in order.

### P32 — Cognitive Shadows

- **Tier 1:** Not every account of persistence beyond death describes the same phenomenon. Some are true Cognitive Shadows. Some are not.
- **Tier 2:** A Cognitive Shadow is a dead person's mind sustained by a Shard's Investiture. Identify which Shard sustains each true Shadow. Use the grid.
- **Tier 3:** Six true Cognitive Shadows. Two impostors (an Awakened object and splinters of a Shard). For each true Shadow, find the row-column intersection in the grid. Read in A-through-H order.

### P33 — Collected Powers

- **Tier 1:** Eight vignettes. Two are false — abilities never confirmed in published record.
- **Tier 2:** The wanderer has Yolish Lightweaving, Allomancy (via lerasium), Fortune, Awakening, BioChromatic Breath, and a Nahel bond. Healing and Forgery are unconfirmed.
- **Tier 3:** Identify the six genuine abilities by their precise Cosmere term. Extract from each term at the marked position. All terms are single words (spaces removed).

### P34 — Letters to a Dragon

- **Tier 1:** Eight letters. Each writer has a voice. The voice reveals the writer.
- **Tier 2:** Six of the eight form a chronological sequence — the Stormlight Archive epigraph correspondence. Two fall outside. Set those two aside.
- **Tier 3:** Identify all 8 writers. The two outside the SA epigraph sequence are the Splintered Shard and the one who refused further correspondence. Extract from the remaining six sender names at the marked positions.

### P35 — The Memory Thief

- **Tier 1:** Six memories are tampered. Two are accurate. Find the errors.
- **Tier 2:** Each tampered memory has one wrong element — a wrong name, metal, term, or faction. Cross-reference against known Cosmere facts.
- **Tier 3:** The wrong elements' first letters, read in order I through VIII (skipping the two accurate memories), spell what was done to the wanderer.

### P36 — All Roads Lead to Hoid

- **Tier 1:** Twelve events. All connect to the wanderer. But the connections are not equal.
- **Tier 2:** Direct means the wanderer chose to act — he spoke, he initiated, he was the agent. Passive presence or being acted upon is indirect.
- **Tier 3:** Identify each event. Classify as direct or indirect. Extract the marked letter from direct events only, reading I through XII. The indirect events produce noise.

---

### META-III — ADONALSIUM

- **Tier 1:** Sixteen Shards. Twelve are represented in the puzzles. Four are missing.
- **Tier 2:** The four missing Shards' first letters anagram to the answer.
- **Tier 3:** Missing: Ambition, Dominion, Invention, Mercy. First letters: A, D, I, M. The thematic clue ("where Hoid has always been") resolves the anagram.

---

### SUPER-META — THE SEVENTEENTH SHARD

- **Tier 1:** Three meta answers. Three Realms. A cycle.
- **Tier 2:** The extraction cycles through the Realms: Physical (Round I), Cognitive (Round II), Spiritual (Round III), Physical (Round I), Cognitive (Round II). Each step goes one position deeper.
- **Tier 3:** Step 1: Round I answer, position 1. Step 2: Round II answer, position 2. Step 3: Round III answer, position 3. Step 4: Round I answer, position 4. Step 5: Round II answer, position 5. Read the five letters.

*One trusts these nudges are sufficient. The answers are never far from the surface. The trick has always been knowing where to look.*
