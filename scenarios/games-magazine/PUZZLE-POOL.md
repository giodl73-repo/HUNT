# Puzzle Pool — Hidden in Plain Sight

**Stage 3: PUZZLE POOL**

14 candidates for 8 slots. Ranked by: puzzle quality, page fit, type diversity, theme alignment ("hidden in plain sight").

---

## Candidates

### Crossword Candidates (for P1, pages 2-3)

**C1. "Lurking Words"** ★★★
15x15 themed crossword where every theme answer contains a smaller word hidden inside it. CHARTER contains HART. THERAPIST contains RAPIST (too dark — use THERAPIST contains THE). PINEAPPLE contains APPLE. Solver must identify the hidden words; their first letters spell a bonus answer.
- Page fit: Perfect for spread (grid + 2 columns of clues)
- Theme alignment: Strong — words literally hidden inside words
- Mechanism: Hidden word identification + first-letter extraction

**C2. "Double Exposure"** ★★★
15x15 crossword where 6 theme entries have dual clues — one for the obvious answer, one for a word concealed within the grid reading diagonally. Standard crossword solving plus a discovery layer.
- Page fit: Good for spread
- Theme alignment: Strong — hidden diagonal words
- Mechanism: Standard solve + diagonal discovery

### Cryptic Candidates (for P2, page 4)

**C3. "Invisible Ink"** ★★★★
10 cryptic clues where every answer is a word meaning "hidden" or "concealed" (LATENT, COVERT, ARCANE, VEILED, BURIED, MASKED, UNSEEN, OCCULT, CLOSET, CIPHER). The cryptic mechanisms themselves demonstrate concealment: containers, hidden words, reversals.
- Page fit: 10 clues fit single page comfortably
- Theme alignment: Perfect — every answer IS the theme
- Mechanism: Standard cryptic with thematic unity

**C4. "Ghost Letters"** ★★★★★
10 cryptic clues where each answer has one letter that does not appear in the clue's wordplay. The 10 "ghost letters" spell a hidden message.
- Page fit: Tight for single page (needs explanation)
- Theme alignment: Very strong — absent letters are hidden
- Mechanism: Meta-cryptic (identify missing letter per clue)

### Logic Grid Candidates (for P3, page 5)

**C5. "The Camouflage Club"** ★★★
5 members of a naturalists' club each study a different animal known for camouflage (chameleon, octopus, stick insect, arctic fox, cuttlefish). Each has a different favorite habitat, observation method, and discovery year. Classic logic grid with a thematic twist: one clue is deliberately misleading (camouflaged among the true clues) and must be identified and discarded.
- Page fit: Good for single page (5x5 grid + 8 clues)
- Theme alignment: Strong — camouflage + one hidden false clue
- Mechanism: Logic elimination + false-clue identification

**C6. "Hidden Rooms"** ★★★
5 historical buildings each have a recently discovered secret room. Match building to room type, year discovered, contents found, and which wall it was behind. Straightforward logic grid with thematic flavor.
- Page fit: Good for single page
- Theme alignment: Moderate — hidden rooms, but straightforward grid
- Mechanism: Standard logic elimination

### Visual Candidates (for P4, pages 6-7)

**C7. "Plain Sight"** ★★
A detailed illustrated scene (market square, library, park) where 12 objects are camouflaged into the environment. Each object's name has a number of letters; the Nth letter of the Nth object (in order of discovery difficulty) spells a phrase.
- Page fit: Perfect for full spread (large illustration)
- Theme alignment: Perfect — hidden objects in plain sight
- Mechanism: Visual search + letter extraction

**C8. "Figure and Ground"** ★★★
A full-spread black-and-white illustration that is simultaneously two images (like Rubin's vase). One image is obvious; the other is hidden. The hidden image contains text that answers the puzzle.
- Page fit: Perfect for spread
- Theme alignment: Perfect — figure-ground concealment
- Mechanism: Perception shift (one aha)

### Word Puzzle Candidates (for P5, page 8)

**C9. "Nested Words"** ★★★★
12 clue phrases where the answer word is hidden consecutively within the phrase text. "The dePICTion was vivid" hides PICT. But the twist: the hidden word is not the answer. The answer is a DIFFERENT word suggested by the remaining letters when the hidden word is removed.
- Page fit: Good for single page
- Theme alignment: Strong — words nested in sentences
- Mechanism: Hidden word identification + transformation

**C10. "Redacted"** ★★★★
A short essay (200 words) about famous acts of concealment in history. 8 words have been "redacted" (blacked out). The lengths of the redacted words are given. From context and cross-referencing clues embedded in the surrounding text, the solver reconstructs the redacted words. Their first letters spell the answer.
- Page fit: Good for single page
- Theme alignment: Perfect — redaction = concealment
- Mechanism: Contextual deduction + extraction

### Mathematical Candidates (for P6, page 9)

**C11. "The Hidden Sequence"** ★★★★★
A 6x6 grid of two-digit numbers. The grid appears random but contains 6 concealed arithmetic sequences (horizontal, vertical, or diagonal). Once all 6 sequences are found, the numbers NOT part of any sequence form a pattern that encodes the answer.
- Page fit: Good for single page (grid + work space)
- Theme alignment: Strong — sequences hidden in noise
- Mechanism: Pattern recognition + elimination

**C12. "Steganography"** ★★★★★
A seemingly straightforward number puzzle (like a KenKen or Kakuro variant) where the completed grid contains a second puzzle: certain cells, when read in a specific order, produce a message. The solver must first solve the number puzzle, then discover the hidden layer.
- Page fit: Tight for single page (needs the grid + rules + work area)
- Theme alignment: Perfect — literal steganography
- Mechanism: Two-layer puzzle (solve + discover)

### Short/Mini Candidates (for P7-P8, pages 10-11)

**C13. "Spot the Difference"** ★★
Two nearly identical short paragraphs. Find the 8 differences. The changed letters spell a word.
- Page fit: Perfect for half page
- Theme alignment: Good — differences hidden in similarity
- Mechanism: Comparison + extraction

**C14. "The Vanishing Act"** ★★★★★
What appears to be a simple word search grid (10x10). But the word list contains 15 words and only 14 can be found. The missing word IS the answer. Except: the missing word IS in the grid — reading backwards, diagonally, in a spiral, or some other non-standard path. The "mini" is deceptively complex.
- Page fit: Word search grid fits full page, intro on half of page 10
- Theme alignment: Perfect — a word that vanishes from a word search
- Mechanism: Exhaustive search + discovery of non-standard reading
