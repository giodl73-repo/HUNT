## Theme: "Hidden in Plain Sight"

Every puzzle in this spread explores concealment in some form. Not as a meta mechanism (there is no meta) but as an editorial thread — the thematic glue that makes 8 independent puzzles feel like one magazine issue.

Concealment takes many forms across puzzle types:
- **Crossword**: hidden words within clue text
- **Cryptic**: double meanings, containers, reversals
- **Logic grid**: the truth hidden behind conflicting statements
- **Visual**: optical illusions, camouflaged objects, figure-ground
- **Wordplay**: words hidden inside other words, anagram cloaking
- **Mathematical**: patterns concealed in sequences or grids

The title page sets this up. The editor's note on page 12 ties it together.

## Page Budget (Law)

Exactly 12 pages. No exceptions. If a puzzle runs long, cut it.

```
Page 1:     Title spread — "Hidden in Plain Sight" editorial intro
Pages 2-3:  P1 — Crossword (15x15, full spread)
Page 4:     P2 — Cryptic clues (10 clues, single page)
Page 5:     P3 — Logic grid (single page)
Pages 6-7:  P4 — Visual puzzle (full spread)
Page 8:     P5 — Word puzzle (single page)
Page 9:     P6 — Mathematical puzzle (single page)
Page 10:    P7 — Short puzzle (half page) + P8 intro (half page)
Page 11:    P8 — Mini puzzle (full page)
Page 12:    Solutions + editor's note
```

Total: 12 pages. 8 puzzles. 0 wasted space.

## Editorial Voice

The magazine editor is the narrator — not a character, but a sensibility. The voice rules:

1. Second person ("you") for puzzle instructions.
2. Present tense in intros.
3. No exclamation marks in puzzle text (the magazine is confident, not excitable).
4. Intros are 40-80 words. No shorter. No longer. Every word earns its space.
5. Difficulty ratings use filled stars (not numbers): ★★ to ★★★★★.
6. "Solution on page 12" at the bottom of every puzzle page.
7. Bylines use fictional author names — the same 4-5 names appear across issues.

## What This Scenario Tests

1. **Page budget as hard constraint** — does the toolkit support fixed page counts?
2. **Editorial format** — can puzzle pages include title, stars, byline, intro, content, solution ref?
3. **Print-first workflow** — no website, no props, no event. Just pages.
4. **Self-contained puzzles** — no props, no QR codes, no online components.
5. **Type diversity** — crossword, cryptic, logic, visual, word, math, mixed.
6. **Publication pipeline** — what changes when the audience is readers, not solvers at an event?

---

## Pipeline Divergences Identified at Scope

1. **No world/ directory needed** — puzzles are self-contained, not built on a shared knowledge base. GAMES Magazine puzzles are about puzzle mechanics, not domain knowledge.
2. **No meta** — replace meta design with editorial coherence review.
3. **No answer security needed** — solutions are printed on page 12. This is a magazine, not a competition.
4. **No narrator character** — the magazine editor's voice is implicit, not a named persona.
5. **Page budget replaces puzzle count** — the constraint is pages, not puzzles. An extra puzzle means a page is too crowded.
