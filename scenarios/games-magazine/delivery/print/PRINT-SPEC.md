## Special Production Notes

1. **Pages 2-3 (crossword spread):** Designed as a facing-page unit. Grid crosses the gutter — ensure registration is tight. Allow 0.125 in overlap at gutter center to prevent white gap.

2. **Pages 6-7 (illustration spread):** Full-bleed illustration. Art must extend to bleed edge on all four sides. Object list sits in a 0.75 in strip along bottom within safety margin. This is the visual centerpiece — highest production priority.

3. **Page 10 (split page):** Horizontal rule at vertical midpoint (5.4375 in from top trim). Top half: P7. Bottom half: P8 intro + word list. Rule weight: 1pt, color #1a1a2e.

4. **Page 12 (solutions):** 3-column layout at reduced font size (8-9pt body, 10pt leading). Editor's note occupies bottom-right column. All solution text must fit without overflow. Proof at actual size.

5. **Crossword grid (P1):** Grid cells 0.28 in square. Total grid: 4.2 x 4.2 in. Centered on the spread. Numbers in top-left of cells at 5pt. Black cells solid fill.

6. **Word search grid (P8):** Grid cells 0.25 in square. Total grid: 3.0 x 3.0 in. Centered on page 11 with solver workspace around margins.
