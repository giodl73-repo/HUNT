## Page 1 — Title + Editorial Intro

**Layout:** 2-column (wide)

```
┌─────────────────────────────────────┐
│           GAMES MAGAZINE            │  ← masthead, 24pt Helvetica Neue Bold
│         Vol. 47 · No. 3            │
│                                     │
│     HIDDEN IN PLAIN SIGHT           │  ← section title, 36pt, dark navy
│                                     │
│  ┌───────────────┬────────────────┐ │
│  │               │                │ │
│  │  Editorial    │  Editorial     │ │  ← 2-column body text, Georgia 10pt
│  │  intro text   │  intro cont.  │ │     ~200 words total
│  │  (col 1)      │  (col 2)      │ │
│  │               │                │ │
│  └───────────────┴────────────────┘ │
│                                     │
│  ┌─────────────────────────────────┐│
│  │ IN THIS SECTION                 ││  ← TOC table
│  │ Page │ Puzzle │ ★ │ Author     ││     8 rows, compact format
│  │ ...  │ ...    │   │ ...        ││
│  └─────────────────────────────────┘│
│                                     │
│                 1                    │  ← page number
└─────────────────────────────────────┘
```

**Notes:** Clean, magazine-editorial. Drop cap on first paragraph optional. The TOC table anchors the bottom third. No puzzle content on this page.

## Page 5 — P3: The Camouflage Club (Logic Grid)

**Layout:** 3-column

```
┌─────────────────────────────────────┐
│ THE CAMOUFLAGE CLUB  ★★★            │
│ By Nadia Koresh                     │
│                                     │
│ [Intro paragraph — ~55 words]       │
│                                     │
│ ─────────────────────────────────── │
│                                     │
│ MEMBERS: Alice, Blake, Carmen, ...  │
│ ANIMALS: Arctic fox, Chameleon, ... │
│ HABITATS: Coral reef, Mangrove, ...│
│ TOOLS: Binoculars, Camera trap, ...│
│                                     │
│ ─────────────────────────────────── │
│                                     │
│ CLUES                               │
│ 1. The person who studies...        │
│ 2. Blake uses the diving mask...    │
│ ... (10 clues)                      │
│                                     │
│ ─────────────────────────────────── │
│                                     │
│ ┌─────────────────────────────────┐ │
│ │ ELIMINATION GRID (5 x 15)      │ │  ← condensed grid, 7pt font
│ │ Members vs all attributes       │ │
│ └─────────────────────────────────┘ │
│                                     │
│ SOLUTION TABLE                      │
│ Member │ Animal │ Habitat │ Tool    │
│ ...    │ ...    │ ...     │ ...     │
│                                     │
│ The false clue is: Clue #____       │
│                                     │
│                  Solution on page 12│
│                 5                    │
└─────────────────────────────────────┘
```

**Notes:**
- This is the tightest single-page layout. The elimination grid (5 members x 15 attributes) is large.
- Use condensed format: grid at 7-8pt, thin rules (0.25pt).
- Categories listed inline (not as separate sections) to save vertical space.
- Clues set at 9pt with minimal leading.

## Pages 6-7 — P4: Plain Sight (Visual Spread)

**Layout:** Full-bleed illustration spread

```
PAGE 6 (left/verso)                    PAGE 7 (right/recto)
┌──────────────────────┐               ┌──────────────────────┐
│                      │               │                      │
│                      │               │    PLAIN SIGHT ★★    │
│                      │               │    By Felix Ortega   │
│                      │               │                      │
│   FULL-BLEED         │               │       FULL-BLEED     │
│   LINE ART           │               │       LINE ART       │
│   ILLUSTRATION       ← gutter →      │       ILLUSTRATION   │
│                      │               │                      │
│   (library interior  │               │    library interior   │
│    with 10 hidden    │               │    continued)         │
│    objects)          │               │                      │
│                      │               │                      │
│                      │               │                      │
├──────────────────────┤               ├──────────────────────┤
│ [Intro — 40 words]   │               │ Find These Objects:  │
│                      │               │ 1. Royalty wears (5) │
│                      │               │ 2. Holds a ship (6)  │
│                      │               │ ... (10 items)       │
│          6           │               │ Solution on page 12 7│
└──────────────────────┘               └──────────────────────┘
```

**Notes:**
- Illustration fills both pages to bleed edge. This is the magazine's visual centerpiece.
- Title, byline, and intro tucked into a 0.75 in strip at bottom-left (page 6).
- Object list in a 0.75 in strip at bottom-right (page 7).
- Bottom strips have white or tinted background to ensure text legibility over art.
- Art must be delivered as 600 dpi line art or vector SVG.
