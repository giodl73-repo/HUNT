# Distribution Plan -- THE GRAND LARCENY

## Per-Team Kit Contents

Each team receives one evidence dossier at check-in. The dossier is a labeled manila folder containing everything needed to solve the hunt.

### Kit Manifest

| Item | Description | Pages/Qty | Stock | Stapled? |
|------|-------------|-----------|-------|----------|
| Manila folder | Letter-size, tab closure | 1 | Standard manila | -- |
| Dossier label | Applied to folder front | 1 label | Adhesive paper | -- |
| Cover sheet | Case briefing (PR02) | 1 page | Cardstock 65lb+ | No |
| Exhibit A -- Police Report | P1 document (PR03) | 2 pages | White 20lb | Yes |
| Exhibit B -- Hotel Receipt | P2 document (PR04) | 2 pages | White 20lb | Yes |
| Exhibit C -- Floor Plan | P3 document (PR05) | 1-2 pages | White 20lb | No |
| Exhibit D -- Witness Statement | P4 document (PR06) | 2 pages | White 20lb | Yes |
| Evidence Correlation Log | Meta worksheet (PR07) | 1 page | White 20lb | No |
| Hotel Key Card | Physical prop (PR08) | 1 card | Cardstock 110lb+, white | -- |
| Calling Card | Physical prop (PR09) | 1 card | Cardstock 110lb+, cream | -- |
| Pens | For solving | 2 | Standard ballpoint | -- |

**Total printed pages per team:** 9-10
**Total loose props per team:** 2 (key card + calling card)
**Total items per folder:** ~12 pieces

## Assembly Instructions

### Preparation (before assembly day)

1. **Print all documents.** See `delivery/print/PRINT-CHECKLIST.md` for file list and page counts.
2. **Print props.** Hotel key cards and calling cards from their respective HTML files.
3. **Print labels.** Dossier folder labels from `delivery/print/props/labels-dossier.html`.
4. **Cut cards.** Hotel key cards to 3.5x2in, calling cards to 3x2in.
5. **Cut labels.** Cut adhesive labels to size.
6. **Staple multi-page documents.** P1 (2 pages), P2 (2 pages), P4 (2 pages) each get one staple, top-left corner.

### Assembly (per team, ~15 minutes)

```
Step 1: Take one manila folder.
Step 2: Write team name on one dossier label.
        Apply label to front of folder.
Step 3: Stack documents in this order (top to bottom):
          a. Cover sheet (cardstock)
          b. Exhibit A -- Police Report (2pp stapled)
          c. Exhibit B -- Hotel Receipt (2pp stapled)
          d. Exhibit C -- Floor Plan (1-2pp loose)
          e. Exhibit D -- Witness Statement (2pp stapled)
          f. Evidence Correlation Log (1pp)
Step 4: Place stack into folder.
Step 5: Drop hotel key card (white, ROOM 303) loose into folder.
Step 6: Drop calling card (cream, 3-1-2-6) loose into folder.
Step 7: Close folder.
Step 8: Verify: folder label shows team name and case number.
```

## Distribution Protocol

### Check-in (start of event)

- Staff at check-in table.
- Each team registers and receives their labeled dossier.
- **Staff script:** "Your case file, detective. Everything you need is inside. Submit your final answer at the website when you've cracked the case."
- No further instructions given. Teams open the dossier and begin.

### During the Hunt

- No triggered deliveries. All materials given at start.
- Staff available for hint requests only.
- Answer submission via website -- staff monitors submissions.

### Emergency Supplies

Keep at the admin table at all times:
- 2 extra copies of every printed document
- 4 extra hotel key cards
- 4 extra calling cards
- 2 extra manila folders with labels
- Spare pens
- A printed copy of each puzzle's solution (for staff reference only)

### End-of-Hunt

- No props to collect. All items are keep-items.
- Run answer reveal / debrief session.
- Optional: provide a "case closed" certificate or stamp for the folder.

## Day-of Timeline

```
T-60 min    Arrive at venue. Set up admin table.
T-45 min    Lay out all dossiers. Verify count matches team count.
T-30 min    Test answer submission website (submit test answer, verify it works).
T-15 min    Staff briefing. Review hint policy. Distribute staff solution copies.
T-0         Doors open. Check-in begins. Dossiers distributed.
T+90 min    First teams likely finishing. Monitor submissions.
T+120 min   Most teams finished. Begin wrap-up.
T+130 min   Answer reveal and debrief session.
T+150 min   Event ends. Breakdown.
```
