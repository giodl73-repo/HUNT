## Solution

### Identification

**Sighting 1:** Glacial Reach. Ice. High DEF. Low SPD. Drops a crest.
- Candidates: Frostclaw (#07, DEF 22), Tundra Golem (#09, DEF 50), Permafrost Knight (#12, DEF 38).
- "Drops a crest" → Knight's Crest → **Permafrost Knight (#12)**.
- Confirm: HP 280, ATK 30, DEF 38, SPD 10, Ice. Drops Knight's Crest. Matches.

**Sighting 2:** Stormspire Peaks. Earth. Highest DEF in the game. Drops an engraved plate.
- Only Earth enemies in Stormspire: Stone Sentinel (#15, DEF 55), Peak Warden (#18, DEF 30).
- "Highest DEF in the game" → DEF 55 → **Stone Sentinel (#15)**.
- Confirm: HP 400, ATK 28, DEF 55, SPD 5, Earth. Drops Runed Plate. Matches "engraved plate."

**Sighting 3:** Iron Citadel. Dark. Highest non-boss ATK. Low DEF. Drops an essence.
- Iron Citadel Dark enemies: Dread Knight (#19, ATK 55), Void Shade (#20, ATK 60), Iron Revenant (#21, ATK 42), Ash Phantom (#23, ATK 52).
- "Highest non-boss ATK" → ATK 60 → **Void Shade (#20)**.
- Confirm: HP 200, ATK 60, DEF 10, SPD 36, Dark. Drops Void Essence. Matches "glowing essence."

**Sighting 4:** Iron Citadel. Dark. Highest non-boss HP. Very high DEF. Drops a mechanical heart.
- "Highest non-boss HP" → HP 500 → **Iron Revenant (#21)**.
- Confirm: HP 500, ATK 42, DEF 48, SPD 8, Dark. Drops Iron Heart. Matches "mechanical heart."

**Sighting 5:** Iron Citadel. Dark. Balanced stats. Drops a dark blade.
- "Standard endgame enemy" with balanced stats and a blade drop.
- Dread Knight (#19): ATK 55, DEF 35, SPD 18. Drops Obsidian Blade.
- "Nothing record-breaking" — not highest ATK (Void Shade), not highest HP (Iron Revenant), not fastest (Ash Phantom). Just solid. → **Dread Knight (#19)**.
- Confirm: Matches "dark-bladed weapon" = Obsidian Blade.

### Extraction

**Answer: LOTUS** (ROT13: YBGHF)

### Extraction Verification

Answer: LOTUS. Puzzle slot: Act II, position 1. Super-meta extracts position 1: LOTUS[1] = L. L in ROT13 = Y. Matches the extraction table.

### Skeptic's Notes

The brief wanted "Nth letter of each enemy name where N = sighting number." That extraction is broken — no enemy name starts with L, so sighting 1 cannot produce the first letter of LOTUS. The bestiary-number-to-A1Z26 extraction is cleaner, fairer, and uses information the solver already has. The aha remains the same: identify the enemy from behavioral clues. The extraction is just math, not indexing into names with awkward position counting. Simpler is better.
