## ODN NODE TOPOLOGY — CURRENT STATE

The ODN connects 15 computer nodes through a series of junctions. Each junction can be ACTIVE (passes data) or PASSIVE (blocks data). Nodes that cannot receive data from Primary Computer are considered OFFLINE regardless of power state.

**Routing rule:** Primary Computer must maintain live data paths to ALL nodes it serves. A node is OFFLINE if every data path between it and Primary Computer passes through at least one PASSIVE junction.
