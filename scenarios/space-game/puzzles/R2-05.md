## SYSTEM STATE REFERENCE — BEZEL CARD

```
SYSTEM STATES — STATUS INDICATOR LEGEND

  ◉  ONLINE   — System running (active or idle)
  ◇  READY    — Power available; can be activated on command
  ◌  STANDBY  — Was running; sleeping; instant wake on command
  ⊘  LOCKED   — Power available; restricted; requires deliberate unlock
  ○  OFFLINE  — No power; upstream dependency not met
  ●  FAULT    — Hardware failure; needs repair
```
