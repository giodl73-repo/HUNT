# WAVELENGTH — World Overview

**Domain:** Real-world pop/rock music, 1970s-1990s
**Mode:** Real-world reference (not fictional — all data must be fact-checked)
**Hook:** Solvers decode messages hidden in actual music data — track listings, chart positions, musical notation, lyrics, artist names

## Tone

Late-night FM radio, circa 1995. The crackle before a vinyl track drops. College radio playlists written in felt-tip pen. The DJ who plays album cuts nobody else would. Everything here is real — real songs, real charts, real musicians. The puzzles live in the gaps between the data.

## Canon Rules

- Every artist, album, song title, chart position, and release year must be verifiable against published sources.
- No invented songs, albums, or chart entries.
- Music theory facts (note names, intervals, key signatures) follow standard Western music theory conventions.
- When multiple sources disagree (e.g., chart positions vary by region), use Billboard Hot 100 (US) as the canonical source.
- Track listings follow the original album release (not reissues or deluxe editions, unless specified).
- Lyrics are quoted from published sources. Paraphrases are acceptable for puzzle cluing but must not distort meaning.

## Verification Protocol

Before any data enters a world/ file:
1. Cross-reference at least one published source
2. If uncertain, tag with [VERIFY]
3. After locking, no changes without impact assessment

## Quality Bar

An author who has never listened to these artists should be able to write a factually correct puzzle using ONLY the world/ files.
